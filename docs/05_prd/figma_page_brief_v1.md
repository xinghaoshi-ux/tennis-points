# U-01 排行榜页

页面用途：H5 公共端排行榜展示，校友可查看当前赛季排名，搜索姓名、筛选院系、查看选手积分明细。

页面从上到下：
1. 顶部标题区
   - 页面标题：`THA 年度赛事积分`
   - 副标题：`当前赛季排行榜`
   - 右侧可显示当前赛季名称
2. 搜索与筛选区
   - 姓名搜索框，支持输入后实时筛选
   - 院系下拉筛选器
   - 可选：清空筛选按钮
3. 当前赛季信息卡
   - 赛季名称
   - 起止时间
   - 当前上榜人数
   - 总积分概览
4. 排行榜列表区
   - 排名序号
   - 选手姓名
   - 院系
   - 总积分
   - 比赛场次/积分来源数
   - 点击整行进入积分明细弹窗
5. 积分明细弹窗
   - 选手姓名
   - 所属院系
   - 分类汇总区
   - 逐条积分记录列表
   - 关闭按钮
6. 空状态区
   - 无搜索结果时显示空状态
   - 文案提示当前无匹配数据

点击关系：
- 点击选手姓名或整行 → 打开积分明细弹窗
- 点击院系筛选器 → 展开院系列表
- 输入姓名 → 实时刷新列表
- 点击清空 → 恢复默认榜单

需要修改：
- 排行榜列表默认只展示前 20 条，后续可继续分页加载
- 明细弹窗内复用列表样式，不额外做复杂图表也可
- 移动端优先，列表应适配窄屏显示

需要新增：
- 无数据空状态
- 选手积分明细弹窗

# A-06 Excel 导入页

页面用途：管理管理员上传赛事结果 Excel，完成解析、预览、确认导入和取消上传。

页面从上到下：
1. 顶部页面标题区
   - 标题：`Excel 导入`
   - 副标题：导入赛事结果
2. 赛事选择区
   - 赛事下拉选择
   - 当前赛事信息提示
3. 文件上传区
   - 拖拽上传框
   - 上传按钮
   - 文件格式提示：仅支持 `.xlsx`
4. 解析进度区
   - 上传状态
   - 解析进度条
   - 状态文案：等待解析、解析中、解析完成、失败
5. 预览表格区
   - 行号
   - 选手姓名
   - 赛事结果
   - 匹配状态
   - 警告说明
   - 勾选是否导入
6. 操作按钮区
   - `确认导入`
   - `取消上传`
7. 结果反馈区
   - 导入成功提示
   - 导入失败提示

点击关系：
- 点击赛事下拉 → 选择目标赛事
- 点击上传框或按钮 → 选择文件上传
- 上传后 → 自动进入解析进度状态
- 解析完成 → 展示预览表格
- 点击 `确认导入` → 进入导入流程
- 点击 `取消上传` → 取消当前上传任务

需要修改：
- 页面核心顺序必须是：选赛事 → 传文件 → 看预览 → 再确认
- 上传区和预览区要明显分开
- 不要把模板下载做成主视觉，本期不实现可置灰隐藏

需要新增：
- 解析进度条
- 预览表格
- 导入结果反馈区
